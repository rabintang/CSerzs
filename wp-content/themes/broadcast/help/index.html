A) Theme Installation - top

Extract the files from the zip folder. There should be a folder called "Broadcast" upload this to your themes folder via FTP. The next step is to activate the theme. You will login into your admin panel. Appearance » Themes » Broadcast click the activate link.
If you need help selecting a ftp. I use FileZilla.

B) Creating Posts - top

I'm going to assume you know how to create a post in wordpress admin panel if not to create a post in wordpress first login into the admin panel » Posts » Add New. Ok, After you type in your title and post content. You'll want to add a image so it displays in the featured slider. The way to do this is to set a featured image.

  1. First Step (Click "Set featured image") Located on the right side of the panel.
  2. Second Step (Click select files and select a image for your featured image.)
  3. Third And Final (After upload is done scroll down and click "Use as featured image")

I strongly recommend you use a image the size of 438 X 332 or bigger. For better resolution in the featured slider.

B PART 2) Unique Options - top

There are 4 Unique options on the right side of your edit or add post page. These options will give you more flexibility. I've listed these items below.

  1. Add To Spotlight Options.

    2. The "Add To Spotlight" Option is for the spotlight located above your slider. It adds the item to a jcarousel slider. You can access the spotlight's settings here. Appearance » Broadcast Options » Home Page » Spotlight.

  2. Sticky Post Option.

    3. The "Sticky Post Option" Is for when you want to select any post and sticky it to your home page. All you have to do is go to your posts in the admin panel, find the post you want to sticky and simply click the option it will now show up on your home page. Note: This option only works for the "Featured Post Option" Located In the options panel under. Appearance » Broadcast Options » Home Page » Blog Posts » Click "Featured Post" And save changes.

  3. Add To Featured Slider Option.

    4. Use the "Add To Featured Slider Option" when you want to add posts to your featured slider.

  4. Set Featured Video Option.

    5. The "Set Featured Video" Option is a quick option to add a featured video to any blog post. Note: The post must be stickied for this to show up on the homepage.

C) Widgets - top

There are 8 custom widgets.

  1. Recent News

    2. This widget will display latest posts. The default listing number is 5. You may change this number to anything you'd like you can also select certain categories to display.

  2. Video Embed

    3. Embed any video you'd like from any major video provider. Simply paste the video embed code into the textarea box to display the video.

  3. Flickr

    4. Display as many as flickr photo's as you'd like with this widget.

  4. 300 x 250 Ad

    5. This widget comes with a "Ad Code" Input field and also a "Image Url" Input for the various types of ads you may want to display.

  5. 125 x 125 Ad

    6. Use this widget to place quick ad's onto your website or advertise web space.

  6. Sidebar Tabs

    7. Use this widget show the sidebar tabs. Popular, Recent, Comments & Tags

  7. Social Icons

    8. This widget will display 4 social icons (Twitter, Facebook, Rss & Vimeo).

  8. Popular News

    9. This widget will display your most popular posts.

To enable these custom widgets simply drag them into one of the widgetized areas listed below.

D) Template Pages - top

There are 3 custom page templates:

  1. Archive

    2. This page will display the last 10 posts in each category

  2. Contact Form

    3. This is a contact page with a fully functional contact form. Note: The contact will not work unless you setup your email in the options panel. Located at Appearance » Broadcast Options » Miscellaneous » Contact Email » Input your email and "Save Changes".

  3. Full Width

    4. This page will display a full-width page with no sidebars.

The page attributes is located on the right side of the add new or edit pages page. These page templates are only for pages not blog posts.

E) Theme Options - top

To access the theme options panel go to Appearance >> Broadcast Options. Once your on the theme options landing page. Select any page that you'd like to edit threw the options. Here are a couple of tips to get the most out of the options panel.

  • General Settings

The general settings will help you edit settings that affect the whole site.

  • Home Page

The home page settings will only affect the home page settings. On this options page you may choose between 2 featured styles. A featured slider and featured post. There is also 2 different blog listing styles available. You may choose to disable the featured slider or post if you'd like to have no featured item.

  • Post Page

The post page is the options for the page 'page.php' also known as the post page any settings changed in this section will affect every page from the blog posts.

  • Miscellaneous

This is where the ad settings will be for the 468 x 60 and 300 x 250 ads. A important note about the "Follow Icons" page. Is you must input your username for the icons to link back to your social media page. Please be sure to add this in if you choose to display the icon.

F) Css Files & Theme Structure - top

Broadcast uses 6 Css Files.

  1. style.css - Main Css File. Controls the whole theme structure.
  2. scripts/css/featured_slider.css - This file is for the featured slider and featured post.
  3. scripts/css/firstnavigation.css - This file is for the top header navigation.
  4. scripts/css/secondnavigation.css - This file is for the categories navigation.
  5. scripts/css/tabs.css - This controls the tabs on the right sidebar.
  6. scripts/css/styles/dark/dark.css - This is the css file for the theme skin dark.

If you want to edit any of these css files the are the locations to the files.

J) Common Problems - top

  • My images aren't working. - If your images aren't working make sure to chmod the files "thumb.php" and the folder "cache" to either 777 or 755.

  • My slider doesn't work. - A lot of users just upload the complete zip file they downloaded from ThemeForest. But the actual way to upload is to open the zip file you downloaded from ThemeForest. And only upload the zip file "Broadcast.zip" or upload the "Broadcast" folder found inside the file you downloaded from ThemeForest. Do not upload all the contents of the zip file you downloaded from themeforest this will only result in javascript errors. If this does not work for you. Then it's a plugin causing the problems you'll have to disable each plugin and re-activate one by one to see which one is causing the problems.
